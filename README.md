My sinatra template
================

My Sinatra template with Bundler, Rack config, Rack Contrib, RSpec2, SimpleCov and some useful Rake tasks.

Requirements
------------
* Ruby ~> 1.9.2

Useful Tasks
------------

* Run Tests `rake spec`
* Run development console `rake console`
* Check Available rake tasks `rake help`
